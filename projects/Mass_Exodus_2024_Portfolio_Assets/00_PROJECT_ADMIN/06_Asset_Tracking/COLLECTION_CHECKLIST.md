- [ ] Branding master document
- [ ] Final poster files
- [ ] Posters installed in context
- [ ] CLO 3D identification renders

- [ ] Floor plans

## Before publication
- [ ] Confirm image rights and participant consent
- [ ] Remove private information from screenshots
- [ ] Verify all numbers and dates
- [ ] Confirm collaborative credits
- [ ] Export desktop and mobile crops
- [ ] Create poster frames for video
- [ ] Create reduced-motion alternatives
- [ ] Compress all public assets
- [ ] Add descriptive alt text
